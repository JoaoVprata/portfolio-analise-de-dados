# Análise de Dados — Clínica de Estética

## Contexto
A clínica possui dados de atendimentos, mas não consegue identificar quais serviços e profissionais geram mais lucro.

## Objetivo
Criar métricas financeiras e operacionais para apoiar decisões estratégicas.

## Dados
Foram utilizados arquivos CSV contendo informações de clientes, agendamentos, financeiro e profissionais.

## Métricas Criadas
- Faturamento
- Lucro
- Ticket médio
- Taxa de faltas
- Lucro por serviço
- Faturamento por profissional

## Principais Insights
- Serviços com alto volume nem sempre geram maior lucro
- A taxa de faltas impacta diretamente o faturamento
- Diferenças de performance entre profissionais

## Ferramentas
- Python (Pandas, Matplotlib)
- Excel
